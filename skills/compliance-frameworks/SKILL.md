# Compliance Frameworks for Urbit Deployments

IMPORTANT: WHILE URBIT'S NETWORKING ARCHITECTURE HAS BEEN AUDITED, THE CURRENT RUNTIME IS NOT ASSUMED TO BE SECURITY HARDENED. BEFORE USING URBIT FOR HIGHLY REGULATED USE CASES, ENSURE YOU PERSONALLY UNDERSTAND THE PRESENT NATURE OF URBIT SECURITY. ANYONE WORKING ON COMPLIANCE-SENSITIVE IMPLEMENTATIONS SHOULD BE MUCH CLOSER TO THEIR UNDERSTANDING OF URBIT'S CAPABILITIES AND THE WAY THAT COMPLIANCE FRAMEWORKS APPLY TO THEIR SPECIFIC BUSINESS.
